# Export to Excel GUI
This GUI lets you export data from the database into an excel file. It was created to help our supply chain team download data about POs easier as it was something they previously had to submit a ticket to us for. So now they can pull it themselves anytime.


There is dummy data hardcoded into the scripts for orders #1, #2 and #3 so you can test the script yourself.

When the GUI appears, enter just the order numbers seperated by commas to export the dummy data.
